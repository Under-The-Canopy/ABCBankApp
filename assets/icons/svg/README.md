# SVG图标文件夹

将您的SVG图标文件放置在这个文件夹中，按照以下命名规范：

## 顶部工具栏图标
- `lottery.svg` - 抽奖图标
- `search.svg` - 搜索图标
- `voice.svg` - 语音图标
- `version.svg` - 版本图标
- `customer-service.svg` - 客服图标
- `message.svg` - 消息图标
